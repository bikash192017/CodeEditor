\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
The shift towards remote work and distributed teams has necessitated the development of advanced collaboration tools, particularly in the realm of software engineering. This project report presents the design, development, and implementation of a Real-Time Collaborative Code Editor. The system is built using the MERN (MongoDB, Express.js, React, Node.js) stack, enhanced with WebSocket for real-time synchronization, WebRTC for peer-to-peer audio/video and screen sharing capabilities, and integrated AI features for code assistance. This report details the architecture, methodology, implementation nuances, and comprehensive evaluation of the application.
